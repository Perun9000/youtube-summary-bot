# Кнопка «🔮» на превью по всему YouTube

**Дата:** 2026-07-04
**Статус:** утверждено

## Проблема

Оверлей-кнопка «🔮» на превью видео инжектится только на странице открытого
ролика: в `injectPreviewButtons()` стоит гейт `if (!extractVideoId()) return;`
(browser-extension/content.js). На главной `https://www.youtube.com/`,
в подписках, поиске и на страницах каналов карточки видео идут без кнопки —
пользователь не может отправить ролик боту, не открыв его.

## Решение

Показывать кнопку «🔮» на любой видимой карточке видео по всему YouTube.
Механика инжекции не меняется: селектор `a[href*="/watch?v="]` + требование
картинки внутри + фильтр видимости через `getClientRects()` уже обходят весь
документ и отсекают скрытые SPA-деревья.

### Изменения (только browser-extension/content.js)

1. **Убрать гейт «только /watch»** из `injectPreviewButtons()`.
2. **Перестроить `onRouteChange()`**: сейчас на не-watch страницах он делает
   ранний `return`, и отложенные сканы превью не планируются вовсе. Станет:
   - главная кнопка «🔮 Summary» (ряд Like/Share) — по-прежнему только на
     `/watch`, на остальных страницах uninject;
   - сканы превью (`injectPreviewButtons`) — планируются на любой странице.
3. **Новые исключения** в `injectPreviewButtons()` рядом с существующим
   `#player, ytd-player`:
   - `ytd-notification-renderer` — миниатюры в дропдауне уведомлений слишком
     мелкие, кнопка мешает;
   - `ytd-video-preview` — hover-плеер карточки (inline-превью при наведении),
     кнопка перекрывалась бы контролами плеера.

### Вне охвата

- Shorts-карточки: ссылки `/shorts/…` селектор не матчит; бот заточен под
  обычные ролики.
- CSS (`content.css`) — класс-based, `position: relative` на хосте, работает
  на любых карточках без изменений.
- `manifest.json` (content script уже матчит `*://*.youtube.com/*`), options.

## Проверка

Вручную в Chrome (расширение перезагрузить через chrome://extensions):

1. Главная: навести на карточку → «🔮» слева сверху; клик открывает
   `t.me/<handle>?start=<video_id>` в новой вкладке, НЕ открывая сам ролик.
2. Ленивая подгрузка: проскроллить главную вниз — кнопки появляются на
   дозагруженных карточках (путь через MutationObserver).
3. Дропдаун уведомлений: кнопок нет.
4. Регрессия `/watch`: главная кнопка в ряду Like/Share на месте, кнопки на
   related-сайдбаре работают.
